

一个基于 [Create React App](https://github.com/facebook/create-react-app)的react练手项目，Node版本13.5.0

主要内容：本地请求代理、区分环境打包、注解支持、scss支持等
### 技术栈：react + react-router + mobx + axios

## 开发环境启动
`npm start`
## 打包命令

* 正式 `npm build`
* 外测 `build:release`
